# 🔔 SMZDM 爆料监控器

> **第一时间获取博主爆料，不再错过好价！**

## 💡 项目初衷

你是否也有这样的困扰？

- 什么值得买 App 的消息推送总是**延迟很久**
- 等你收到推送时，**好价爆料已经过期了**
- 特别是像**信小兔**这种爆料快、优惠短的博主，等你看到的时候，优惠券早就抢光了

这个插件就是为了解决这个问题而生——**实时监控博主爆料页面，第一时间推送到企业微信**，让你抢占先机！

## ✨ 功能特点

| 功能 | 说明 |
|------|------|
| ⏱️ 实时监控 | 可配置刷新间隔，最快 30 秒检查一次 |
| 📢 企微通知 | 检测到新爆料立即推送到企业微信群 |
| 🤖 反爬策略 | 支持随机延迟、模拟人类行为，降低触发风险 |
| ⚠️ 验证码检测 | 自动检测验证码并暂停监控，发送警报提醒处理 |
| 📊 详细日志 | 完整的运行日志，方便排查问题 |
| 🎯 精准抓取 | 只推送新增爆料，避免重复通知 |

## 📦 安装

### 方式一：本地安装（开发者模式）

1. 下载或克隆本项目
2. 打开 Chrome/Edge 浏览器，访问 `chrome://extensions/` 或 `edge://extensions/`
3. 开启「**开发人员模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择 `smzdm-monitor-extension` 文件夹

## 🚀 快速开始

### 1. 获取企微机器人 Webhook

1. 在企业微信群中添加「群机器人」
2. 获取机器人的 Webhook 地址（格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx`）

### 2. 配置插件

1. 点击浏览器工具栏中的插件图标 🔔
2. 填写配置：
   - **监控 URL**：目标博主的爆料页面地址（如 `https://zhiyou.smzdm.com/member/信小兔/baoliao/`）
   - **Webhook 地址**：企微机器人的 Webhook URL
   - **刷新间隔**：建议 60-120 秒（太快可能触发反爬）

### 3. 开始监控

点击「**开始监控**」，插件会自动：
- 打开监控页面（后台标签页）
- 定时刷新并检查新爆料
- 发现新爆料时推送到企微群

## ⚙️ 高级设置

### 反爬虫策略

| 模式 | 说明 | 推荐场景 |
|------|------|----------|
| 随机模式 | ±30% 波动 | 日常使用 |
| 保守模式 | 2-4 倍间隔 | 频繁触发验证码时 |
| 激进模式 | 0.5-1 倍间隔 | 紧急监控（风险高） |
| 固定模式 | 严格按设定间隔 | 测试调试用 |

### 验证码检测

当检测到验证码时：
1. 自动暂停监控
2. 发送企微警报提醒
3. 手动完成验证后点击「验证已处理」恢复

## 📋 注意事项

1. **合理设置间隔**：建议不低于 60 秒，避免触发反爬机制
2. **验证码处理**：出现验证码需在 5 分钟内手动处理
3. **后台运行**：监控期间请勿关闭浏览器，可最小化
4. **仅供学习**：本项目仅供学习研究，请遵守网站 ToS
